Вариант 9. Задача FIND.
Идея: создаём два массива, "бегаем" рекурсивно по файлам (как в ls). Сохраняем имена переменных и их директории.
После того, как закончили поиск, отсортировываем массив. Выводим те имена, которые повторяются (задача 1).

Файлами будем считать только регулярные файлы и папки (сделать так, чтобы проверялись все файлы, не сложно: там, где сравнения с типом файла, нужно немножко изменить:
Блок, относящийся к папке, оставить в if'е (сравнение с 4), добавить к этому if'у else, в который вставить код, который сейчас находится в сравнении с 8 (и, соответсветственно, удалить этот if).
Тогда для папок будет вызываться рекурсия, а остальные файлы будут просто вноситься в "базу".)
